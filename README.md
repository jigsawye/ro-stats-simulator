Build with React, Redux, antd, styled-components.

## Build Setup

``` bash
# install dependencies
yarn # or `npm install`

# serve with hot reload at localhost:8080
yarn dev # or `npm run dev`

# build for production with minification
yarn build # or `npm run build`
```
